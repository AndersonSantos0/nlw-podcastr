## Usage

### Get the project

```
$ git clone https://github.com/AndersonSantos0/electron-podcastr.git
```

### Install Dependencies

```
# using yarn
$ yarn

# using npm
$ npm install
```

### Use it

```
# development mode
$ yarn dev or npm run dev

# production build
$ yarn build or npm run build
$ yarn start or npm run start
```
